# XC-OS
> 面向MCU的小型嵌入式图形操作系统

## 内核
> [FreeRTOS Kernel V10.2.0](https://www.freertos.org)

## 图形引擎
> [LittlevGL V6.0](https://github.com/littlevgl/lvgl)

## 特点
> 1.无需外扩**RAM**和**FLASH**
>
> 2.支持安装第三方APP以及驱动程序，使用Lua脚本语言编写
>
> 3.支持总线式扩展外部设备
>
> 4.系统自动识别新设备，安装驱动程序，并生成图形界面
## APPs
> 1.系统设置(显示器亮度、电池信息、电源)
>
> 2.Lua代码编辑器+调试终端
>
> 3.APP启动器
>
> 4.文件浏览器
>
> 5.文本编辑器
>
> 6.音乐播放器
>
> 7.待续...
